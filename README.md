# Automated Bottle Filler System

An automated bottle filling system that runs in the Command Line Interface (CLI).

---

## Inputs

* **Water Bottles:** The user should provide the capacity of each water bottle.
* **Tap's pressure:** The flow rate of each tap should also be given by the user.
* **Input validation:** Every input is validated to be a valid integer greater than 0.
* **Limiters:** Water bottles with capacity uppon 5L will not be allowed by the system. The taps pressure must be between 50ml/s and 250ml/s.

---

## Setup & Installation

1. **Clone the repository:**
```bash
   git clone [https://github.com/gabrielcastro-dev/bottle-filler-system.git](https://github.com/gabrielcastro-dev/bottle-filler-system.git)
```

2. **Install the dependencies:**

In order to install the dependencies you should have Node JS installed on your computer (version 21 is recommended). Then run the command below:

```bash
  npm install
```

## Running the code

With Node JS and the required dependencies already installed, you can run the project by typing:

```bash
  node script.js
```
